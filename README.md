# zork-with-combat
A zork-style text adventure videogame with combat.

## Current Development Status
### **Under development**  
###### *Alpha*  
###### *Beta*  
###### *Release*  
If you would like to help, or just suggest ideas, please feel free to contact me.

## How to build
Thus far, I've been using g++ (gcc) to compile, and I therefore recommend compiling with it.  
Run `g++ -lncurses -std=c++23 main.cpp` in the directory to build.

## Compatability
### Known compatable with:
* macOS arm64
* Linux x86/x86_64
### Should be compatable with:
* Windows x86/x86_64
* Windows arm64
____  
In case of incompatability with any of these platforms, please create an issue and I'll look into it.
